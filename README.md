## рҹҡҰ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/eye-to-eye-app.git
cd eye-to-eye-app
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Backend Server

```bash
cd backend
npm start
```

### 4. Start Frontend

```bash
cd frontend
npm start
```

Your app should now be running on **[http://localhost:3000](http://localhost:3000)** рҹҺү
